# Get Started

DataTable based on https://material-ui.com/api/table/ table component with additional features like search, filtering, sorting and much more.

## Installation
To install material-table with `npm`:

    npm install material-table --save

To install material-table with `yarn`:

    yarn add material-table


> Add material icons url to your html:
```html
<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">
```

## Import

```js
import MaterialTable from 'material-table'
```
